# CordisAI — AI-диагностика эхокардиографии

> Видео УЗИ сердца → 21 измерение → 7+ диагнозов → протокол МЗ РК в 1 клик за 60 секунд.

## Что делает
Врач загружает видео УЗИ сердца (AVI / DICOM / NIfTI) — CordisAI сегментирует камеры нейросетью U-Net, рассчитывает 21 клинический параметр (ФВ, КДО, ИММЛЖ, ГЛС), детектирует 7+ патологий (ХСН, ГЛЖ, амилоидоз, увеличение ЛП) и формирует **стандартизированный протокол МЗ РК** в PDF, FHIR R4 и DICOM SR — готовый к подписи врача и загрузке в МИС.

## Проблема
- Дефицит кардиологов УЗД: в РК на 1 врача — 15 000 человек (норма ВОЗ — 5 000)
- Расшифровка эхо вручную = 20–40 минут, усталость → ошибки
- Нет стандарта протокола → проблемы с аудитом МЗ РК и отказы ФСМС
- 50% случаев ХСН упускаются в первичном звене

## Преимущества
- **× 5 пропускной способности** УЗИ-кабинета (30 мин → 3 мин на исследование)
- **Протокол сразу по МЗ РК** — поля подписи, ссылки на приказы и клинпротокол
- **Не заменяет врача** — AI-ассистент, все значения редактируемы
- **PACS / DAMUMED / FHIR** интеграции — on-premise, данные не покидают клинику
- **Локализация KZ** — кириллица, ИИН, ОСМС, приказы МЗ РК зашиты
- **Ценовое преимущество:** $499/мес vs $2 000+ у us2.ai / EchoGo
