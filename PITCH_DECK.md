### Сердечно-сосудистые заболевания — убийца №1 в мире

**20,5 млн** смертей в год от ССЗ (WHO, 2024) — **32% всех смертей** в мире

**В Казахстане проблема критичнее:**

**Главная проблема:** Эхокардиография — золотой стандарт диагностики сердца, но её интерпретация требует высококвалифицированного специалиста, которых катастрофически не хватает.

## Слайд 3 — Наше решение

### CordisAI — ИИ-кардиолог в каждой клинике

```
Загрузка ЭхоКГ-видео (.avi / .dcm)
        |
        v
   ИИ-сегментация (U-Net) — автоматическое выделение камер сердца
        |
        v
   20+ измерений — LVEF, объёмы, толщина стенок, геометрия ЛЖ
        |
        v
   Выявление заболеваний — HFrEF, HFmrEF, ГЛЖ, дилатация, амилоидоз
        |
        v
   Готовый отчёт — PDF / FHIR / DICOM SR за < 30 секунд
```

**Что делает CordisAI:**
- Загрузка видео ЭхоКГ (стандартный формат с любого аппарата)
- Автоматическая сегментация левого желудочка, миокарда и левого предсердия
- Вычисление **21 клинического показателя** (LVEF, объёмы, толщина стенок, масса ЛЖ, индексированные значения по BSA, GLS)
- Определение **7 патологий** (сердечная недостаточность, гипертрофия, дилатация, амилоидоз, диастолическая дисфункция, клапанные риски)
- **Клинические рекомендации** — ИИ подсказывает врачу дальнейшие шаги (обследования, консультации, терапия по ESC/AHA)
- Генерация стандартизированного отчёта в формате PDF, FHIR R4, DICOM SR
- Результат — за **30 секунд** вместо **20–30 минут** ручного анализа

## Слайд 4 — Как мы обучили ИИ

### 3 нейронные сети, обученные на академических датасетах

**1. U-Net сегментация (основная модель)**
- Архитектура: U-Net 2D (encoder-decoder с skip connections)
- Датасет: **CAMUS** (Университет Лиона, Франция) — 500 пациентов, экспертная разметка
- 4 класса: фон / левый желудочек / миокард / левое предсердие
- **Результат: Dice Score 0.9153** (совпадение с разметкой врача 91.5%)
- Для сравнения: State-of-the-Art в академических публикациях — 0.90–0.93

**2. EF Regressor (предсказание фракции выброса)**
- Архитектура: EfficientNet-B0 (временная регрессия по видеокадрам)
- Датасет: **EchoNet-Dynamic** (Stanford University) — 10 030 видео, 4-камерная проекция
- **Результат: MAE 5.60%** (средняя ошибка ±5.6 процентных пунктов)
- Межоператорская вариабельность врачей: ±5–10%
- Статус: дообучение продолжается (эпоха 11/30, ожидаемый MAE < 5.0%)

**3. View Classifier (определение проекции)**
- Архитектура: EfficientNet-B0 (бинарная классификация)
- Данные: EchoNet A4C + CAMUS A4C/A2C (8 265 видео)
- **Результат: Accuracy 93.3%**

**Вывод для инвестора:** Точность CordisAI на уровне врача-специалиста, валидирована на признанных международных датасетах (Stanford, University of Lyon).

### AI-рекомендации для врача

CordisAI генерирует **персонализированные клинические рекомендации** на основании выявленных находок:
- Дальнейшие обследования (КТ, МРТ сердца, коронарография)
- Лабораторные тесты (NT-proBNP, свободные лёгкие цепи, иммунофиксация)
- Медикаментозная терапия (GDMT: ACEi/ARNi, бета-блокаторы, MRA, SGLT2i)
- Мониторинг (Холтер, амбулаторное АД, сроки повторной ЭхоКГ)
- Консультации специалистов (электрофизиолог, центр амилоидоза)

**Все пороговые значения** — гендер-специфичные (M/F), по стандартам ASE 2015.

### Подключение к УЗИ-аппарату в реальном времени

CordisAI поддерживает **прямое подключение к ультразвуковому аппарату** через DICOM-поток или видеозахват:
- **AI-ассистент во время сканирования** — подсказки по позиционированию датчика, контроль качества изображения
- **Моментальные измерения** — параметры рассчитываются в реальном времени, без ожидания записи
- **Автозахват ключевых кадров** — ИИ автоматически фиксирует ED/ES фазы для точных измерений
- **Уведомления о патологии** — мгновенный alert если обнаружены отклонения во время исследования

Это превращает CordisAI из инструмента пост-обработки в **реального ИИ-ассистента кардиолога**.

### Unit Economics — LTV / CAC

```
  ┌─────────────────────┐         ┌─────────────────────┐
  │                     │         │                     │
  │      доход*         │         │      расход         │         LTV / CAC
  │                     │         │                     │
  │   LTV $12 475       │    –    │   CAC $500          │    =      25x
  │   25 МЕС            │         │   COGS $1 000       │
  │                     │         │                     │
  └─────────────────────┘         └─────────────────────┘
```

**Расчёт:**
- **ARPU** = $499/мес (средний чек по B2B-клиникам)
- **Churn rate** = 4% в мес (отток клиентов)
- **Средний срок жизни клиента** = 1 / 0.04 = **25 месяцев**
- **LTV** (Lifetime Value) = $499 × 25 = **$12 475** — столько денег приносит одна клиника за всё время работы с нами
- **CAC** (Customer Acquisition Cost) = **$500** — стоимость привлечения одной клиники (маркетинг + продажи)
- **COGS** (себестоимость за весь срок) = $40/мес × 25 = **$1 000** — затраты на GPU-инференс и поддержку
- **Чистая прибыль с одного клиента** = $12 475 − $500 − $1 000 = **$10 975**
- **LTV / CAC = $12 475 / $500 = 25x** — каждый вложенный $1 в привлечение возвращает $25
- **Payback** = CAC / ARPU = $500 / $499 = **1 месяц** — клиент окупается за 1 месяц

> Для сравнения: средний LTV/CAC в B2B SaaS — 3–5x. У CordisAI — **25x**, потому что COGS почти нулевой (GPU-инференс стоит $0.08 за анализ), а продукт критически нужен клиникам.

## Краткое описание стартапа

**CordisAI** решает критическую проблему дефицита кардиологов и сонографистов в Центральной Азии и СНГ. Сегодня 55% смертей в Казахстане вызваны сердечно-сосудистыми заболеваниями, при этом на 20 миллионов населения приходится менее 1 000 обученных специалистов по эхокардиографии. Пациенты в районных поликлиниках и сельских клиниках ждут расшифровку ЭхоКГ от 24 до 72 часов — или вовсе не получают её.

CordisAI — это AI-платформа, которая автоматически анализирует эхокардиографическое видео: сегментирует структуры сердца нейронной сетью (U-Net, Dice 91.5%), вычисляет 25+ клинических показателей по стандартам ASE 2015 с гендер-специфичными нормами, выявляет 7+ патологий (сердечная недостаточность, гипертрофия, дилатация, амилоидоз и др.) и генерирует персонализированные клинические рекомендации — всё за 30 секунд вместо 20–30 минут ручной работы.

В отличие от конкурентов (us2.ai, Caption Health, Ultromics), которые работают исключительно на западных рынках и стоят $50–200 за анализ, CordisAI — единственное решение, разработанное специально для региона СНГ: русскоязычный интерфейс, интеграция с местными медицинскими системами (DamuMed, ОСМС), доступная ценовая модель ($3–5 за анализ) и возможность on-premise развёртывания в клиниках без интернета. Ни один из мировых игроков не присутствует в Центральной Азии — мы занимаем этот рынок первыми, превращая экспертный кардиологический анализ из привилегии столичных центров в доступный инструмент для каждой клиники страны.
